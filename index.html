Consejos Generales

Principios fundamentales para formular prompts efectivos con Claude.

1

Sé claro y específico

Indica tu tarea al inicio. Proporciona contexto y divide tareas complejas en pasos manejables.

2

Usa ejemplos

Muestra el formato o estilo deseado. Un buen ejemplo vale más que mil descripciones.

3

Estimula el razonamiento

Pide a Claude que "piense paso a paso" en tareas complejas para obtener respuestas más precisas.

4

Refinamiento iterativo

Si la primera respuesta no es perfecta, pide ajustes específicos. La colaboración mejora los resultados.

5

Aprovecha el conocimiento

Claude tiene amplio conocimiento. Incluye contexto relevante para obtener respuestas dirigidas.

6

Usa role-playing

Pide a Claude que adopte un rol específico para explorar perspectivas y obtener análisis más ricos.

Ejemplo — Ser claro y específico
Prompt débil
"Ayúdame con una presentación."
Prompt efectivo
"Necesito crear una presentación de 10 diapositivas para nuestra reunión de ventas trimestral. Debe cubrir el rendimiento de ventas del Q2, los productos más vendidos y los objetivos de ventas para el Q3. Proporciona un esquema con los puntos clave para cada diapositiva."
El buen prompt especifica el número de diapositivas, el propósito y los temas clave a cubrir.
Ejemplo — Role-playing
Prompt débil
"Ayúdame a prepararme para una negociación."
Prompt efectivo
"Eres un proveedor de telas para mi empresa de mochilas. Me estoy preparando para negociar una reducción de precios del 10%. Como proveedor, proporciona: 1) Tres posibles objeciones, 2) Un contraargumento para cada una, 3) Dos propuestas alternativas. Luego cambia de rol y aconseja cómo abordar mejor esta negociación como comprador."
El role-playing permite explorar múltiples perspectivas y enriquece el análisis.

Creación de Contenido

Técnicas para generar contenido de alta calidad adaptado a tu audiencia.

1. Especifica tu audiencia

Decirle a Claude quién leerá el contenido cambia completamente el resultado. Incluye nivel de conocimiento, intereses y contexto del lector.

  • ¿Es técnico o no técnico?
  • ¿Qué edad o industria?
  • ¿Qué problema están tratando de resolver?
2. Define el tono y estilo

Describe el tono deseado: profesional, casual, humorístico, inspirador. Si tienes una guía de estilo de marca, menciona los puntos clave.

3. Define la estructura de salida

Proporciona un esquema básico o lista de puntos a cubrir. Indica longitud deseada, número de secciones y cualquier elemento específico como llamadas a la acción.

Ejemplo — Descripción de producto
Prompt débil
"Escribe una descripción de producto."
Prompt efectivo
"Escribe una descripción para nuestra nueva silla de oficina ergonómica. Tono profesional pero atractivo. La voz de la marca es amigable, innovadora y orientada a la salud. Incluye: 1) Características ergonómicas clave, 2) Beneficios para la salud y productividad, 3) Mención breve de materiales sostenibles, 4) Llamada a la acción. Aprox. 200 palabras."
Proporciona tono, estilo y elementos específicos para incluir en la descripción.

Resumen de Documentos y Q&A

Cómo extraer el máximo valor de documentos adjuntos.

Consejo clave: Siempre referencia los documentos por nombre, pide citas específicas y define el formato del resumen (número de párrafos, secciones).
Ejemplo — Resumen de informe
Prompt débil
"Resume este informe para mí."
Prompt efectivo
"He adjuntado un informe de 50 páginas llamado 'Tech Industry Trends 2023'. Proporciona un resumen de 2 párrafos enfocado en IA y machine learning. Luego responde: 1) ¿Cuáles son las 3 principales aplicaciones de IA en negocios este año? 2) ¿Cómo impacta el ML en los roles laborales? 3) ¿Qué riesgos menciona el informe sobre adopción de IA? Cita secciones o números de página específicos."
Especifica el enfoque del resumen, preguntas concretas y pide citas para respuestas más verificables.

Análisis de Datos y Visualización

Estructura tus solicitudes de análisis para obtener insights accionables.

Ejemplo — Análisis de ventas
Prompt débil
"Analiza nuestros datos de ventas."
Prompt efectivo
"He adjuntado 'Sales Data 2023'. Presenta los hallazgos así: 1) Resumen ejecutivo (2-3 oraciones), 2) Métricas clave: ventas por trimestre, categoría top, región de mayor crecimiento, 3) 3 tendencias notables con explicación, 4) 3 recomendaciones basadas en datos con justificación. Sugiere 3 visualizaciones para comunicar estos hallazgos."
Estructura clara, métricas específicas y solicita visualizaciones para presentar bien los datos.

Brainstorming

Genera ideas estructuradas y accionables con parámetros claros.

Ejemplo — Actividades de equipo remoto
Prompt débil
"Dame ideas para team building."
Prompt efectivo
"Necesito ideas de team building para un equipo remoto de 20 personas. Sugiere 10 actividades virtuales que promuevan la colaboración. Para cada una: explica cómo fomenta el trabajo en equipo, indica si es mejor para romper el hielo, mejorar comunicación o resolución de problemas. Sugiere una opción económica y una premium."
Parámetros específicos (número de ideas, criterios de categorización) producen resultados más estructurados y útiles.
Tip: Pide respuestas en formatos específicos como viñetas, listas numeradas o tablas para facilitar la lectura y comparación.

Solución de Problemas y Maximizar Rendimiento

Cómo minimizar errores y obtener el mejor desempeño de Claude.

Permitir que Claude reconozca la incertidumbre

Dile a Claude que está bien admitir que no sabe algo: "Si no estás seguro, está bien decirlo." Esto reduce las alucinaciones y mejora la confiabilidad de las respuestas.

Divide las tareas complejas

Si una tarea es demasiado grande y Claude omite pasos, divídela en subtareas más pequeñas y trabájalas una a la vez en mensajes separados.

Incluye todo el contexto en cada conversación nueva

Claude no retiene información de conversaciones anteriores. Siempre incluye el contexto necesario en cada nueva sesión.

Ejemplos Completos

Casos de uso que combinan múltiples técnicas de prompting.

Ejemplo Completo 1 — Estrategia de Marketing
Prompt débil
"Ayúdame a crear una estrategia de marketing."
Prompt efectivo
"Como consultor senior de marketing, necesito tu ayuda para desarrollar una estrategia completa para nuestra nueva línea de accesorios para smartphones eco-amigables. Audiencia objetivo: millennials y Gen Z conscientes del medio ambiente. La estrategia debe incluir: 1) Análisis de mercado (tendencias, 2-3 competidores, tamaño del mercado), 2) Persona del cliente ideal, 3) Marketing mix (producto, precio, plaza, promoción con 5 canales y 3 ideas de campaña), 4) Estrategia de contenido (5 temas, tipos de contenido), 5) KPIs y herramientas de medición. Identifica desafíos potenciales y estrategias de mitigación."
Combina asignación de rol, desglose de tareas, solicitud de estructura, brainstorming y pedido de análisis de riesgos.
Ejemplo Completo 2 — Análisis de Informe Financiero
Prompt débil
"Analiza este informe financiero."
Prompt efectivo
"He adjuntado 'Q2_2023_Financial_Report.pdf'. Actúa como un CFO experimentado y prepara un informe para la junta directiva con: 1) Resumen ejecutivo (3-4 oraciones), 2) Rendimiento financiero (ingresos, márgenes, flujo de caja), 3) KPIs en formato tabla, 4) Análisis por segmento, 5) Balance general, 6) 3 predicciones para Q3, 7) Evaluación de riesgos, 8) Comparación con 2-3 competidores. Genera 5 preguntas que los miembros de la junta podrían hacer, con respuestas sugeridas. Finaliza con un párrafo de apertura para la reunión."
Combina role-playing, análisis estructurado, análisis predictivo, evaluación de riesgos y anticipación de preguntas.
